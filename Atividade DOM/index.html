Atividade DOM

Parte 1 — Pesquisa

1. O que é DOM?
É a estrutura que o navegador cria a partir do HTML, permitindo que o JavaScript acesse e modifique os elementos da página.
2. O que é um nó (node)?
É qualquer parte do DOM: elementos (tags), atributos (id, class) e textos.
3. Métodos: getElementById(), querySelector(), querySelectorAll()
getElementById: seleciona pelo id
querySelector: seleciona o primeiro elemento (CSS)
querySelectorAll: seleciona todos os elementos
4. Diferença entre innerHTML e innerText
innerHTML interpreta HTML
innerText mostra apenas texto
5. O que são eventos?
São ações do usuário como clique (onclick), mudança (onchange) ou passar o mouse (onmouseover).
